# Restaurant System

This is a simple API to register Restaurants and their Dishes. It's made to interact with any front end.

## Requirements

* Ruby 2.3.1

* Rails 5.1.1

* PostgreSQL

## To run locally

* Fork the repo

* `git clone git@github.com:<yourusername>/restaurant-system-api.git`

* `bundle install`

* `rails s`
